\section{Introducción y Objetivos}
En este trabajo de laboratorio se aborda el diseño de un transformador de alta frecuencia utilizado en circuitos tipo fuentes conmutadas, integrando herramientas de IA avanzadas como asistencia al cálculo. El objetivo fundamental es consolidar los conocimientos teóricos de la materia para diseñar el componente a partir de una lista de requerimientos (Grupo 2), garantizando que el flujo magnético máximo no sature el núcleo. Además, se busca analizar críticamente los resultados obtenidos por la IA, justificar técnicamente cada decisión de diseño adoptada y generar una memoria de cálculo completa que culmine con la elaboración del modelo equivalente del transformador, para su posterior ensamble y prueba por parte del grupo ``cliente''.

\section{Especificaciones de Diseño}
Para el desarrollo de este trabajo práctico, se nos asignaron las siguientes especificaciones correspondientes al Grupo 2:
\begin{itemize}
    \item Tensión de entrada ($V_{in}$): 5 V
    \item Frecuencia ($F$): 75 kHz
    \item Ciclo de trabajo ($D$): 2/3
    \item Relación de transformación ($N_2/N_1$): 1.5
    \item Número de secundarios (\# $N_2$): 3
    \item Densidad de corriente admisible ($J$): 3 A/mm$^2$
    \item Corriente continua de magnetización ($IDCN$): 1.1 A $\pm$ 10\%
\end{itemize}
Como restricciones físicas de hardware, se debe utilizar un núcleo de ferrite RM10 (material N30 de TDK con $B_{max} = 100 \text{ mT}$) y un carrete plástico de dos secciones, separando el primario de los secundarios.
